# RecurKit

**Dead simple recurring task manager for solo operators.**

No projects. No boards. No teams. Just the stuff you need to do — on repeat.

---

## What is this?

RecurKit is a lightweight recurring task manager built for freelancers, one-person businesses, landlords, side-hustlers, and anyone who has things that need to happen on a regular schedule.

Examples:

- "Send invoice on the 1st"
- "Check inventory every Thursday"
- "Renew domain March 15"
- "Back up files every Sunday"

You don't need Notion, Asana, or Monday for that. You need RecurKit.

---

## Features

- Daily, weekly, monthly, yearly, and one-time tasks
- Auto-calculates the next due date when you complete a task
- Overdue task highlighting
- Filter by today, upcoming, overdue, or all
- Quick stats dashboard
- Swipe-to-delete on mobile
- Dark mode (follows system preference)
- Multi-language support: English, Turkish, German, French
- Works offline (PWA with service worker caching)
- No sign-up, no accounts, no cloud — data stays in localStorage on your device
- Android app via Capacitor (APK available in Releases)

## Getting Started

### Run in browser

No build step needed. Just open `index.html` in a browser, or serve it locally:

```bash
npx http-server . -p 8080
```

### Build Android APK

Prerequisites: Node.js, Java 17+, Android SDK.

```bash
npm install
node build-www.js
npx cap sync android
cd android
./gradlew assembleRelease
```

The APK will be at `android/app/build/outputs/apk/release/`.

To install on a connected device:

```bash
adb install -r android/app/build/outputs/apk/debug/app-debug.apk
```

## How It Works

1. **Add a task** — give it a name and set the frequency (daily, weekly, monthly, yearly, or one-time).
2. **Check it off** — when done, tap the circle. RecurKit calculates when it's next due.
3. **Stay on top of it** — open the app each day. Overdue tasks are highlighted so nothing slips through.

All data is stored in your browser's localStorage. Nothing is sent anywhere.

| Code | Language |

| EN   | English  |
| TR   | Turkish  |
| DE   | German   |

The selected language is saved in localStorage and persists across sessions.
